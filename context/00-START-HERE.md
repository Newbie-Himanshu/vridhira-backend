## 🚨 COMMON MISTAKES TO AVOID

### Mistake 1: Not Loading Project Rules in Antigravity
```
❌ Start coding immediately
✅ Load .antigravity/rules.md first
```

### Mistake 2: Skipping Docker for Development
```
❌ Install PostgreSQL directly on laptop
✅ Use docker-compose.yml from india-deployment-payment-guide.md
```

### Mistake 3: Testing Payments in Production
```
❌ Use live Razorpay keys immediately
✅ Test thoroughly with test keys first
```

### Mistake 4: Not Following Templates
```
❌ Write services from scratch
✅ Copy templates from antigravity-medusa-quickref.md
```

### Mistake 5: Ignoring Error Handling
```
❌ Basic try-catch
✅ Use robust error handling from india-deployment-payment-guide.md
```
